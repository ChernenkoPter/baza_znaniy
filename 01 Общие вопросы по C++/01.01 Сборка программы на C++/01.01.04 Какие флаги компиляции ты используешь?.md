Как правило руками не собираю, для этого есть CMake, но вот список опций, которые часто применяются
Основные:
- **-v** - подробный вывод
- **-o**  abc - назвать файл объектного кода abc
- **-g** - добавить отладочную информацию
- **-O0, -O1, -O2, -O3, -Os** - задать уровень оптимизации
- **-Wall** - включить основные предупреждения: наиболее важные и общепринятые предупреждения о потенциальных ошибках (неоднозначности, неиспользуемые переменные, присваивание вместо сравнения в if, несоответствия типов и т.д.)
- **-Wextra** - включить дополнительные предупреждения (сравнение переменных типов signed и unsigned, пустое тело оператора)
- **-Werror** - превратить предупреждения в ошибки
- **-std=c++st** - задать стандарт языка **st**
- **-Idir** - добавить директорию **dir** для поиска заголовочных файлов (это заглавная **i**, а не **l**) 
- **-Ldir** - добавить директорию **dir** для поиска библиотек
- **-llib** - линковаться с библиотекой **lib**
- **-Dmacro** — определить макрос **macro**

Более специфичные:
- **-c** - только компиляция без линковки (в объектный файл)
- **-S** - только компиляция в ассемблерный код
- **-E** - только препроцессинг
