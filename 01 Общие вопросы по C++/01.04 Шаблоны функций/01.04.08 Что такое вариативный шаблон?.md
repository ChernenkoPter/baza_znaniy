Вариативный шаблон - шаблон принимающий произвольное количество параметров (типов и значений)
Пакет параметров должен быть в конце списка параметров
```
template<typename... Args>
void printAll(Args... args) {
    (std::cout << ... << args) << std::endl;
} // printAll(1, 2.5, "hello", 'c');

template<auto... Values> // non-type параметры
struct IntegralList {
    void print() { ((std::cout << Values << " "), ...); }
}; // IntegralList<1, 2, 3> il; il.print();
```
