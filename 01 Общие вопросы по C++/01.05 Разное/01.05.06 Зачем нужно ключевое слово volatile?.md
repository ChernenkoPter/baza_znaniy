Указывает компилятору, что данная переменная может быть изменена в другом месте, из-за чего запрещена оптимизация, связанная с удалением или перестановкой обращений к данной переменной
```
volatile char* statusRegister = (char*)0x1234; // адрес регистра устройства
// Ждём, пока устройство не выставит бит готовности
while ((*statusRegister & 0x80) == 0) {
    // Без volatile компилятор мог бы прочитать регистр один раз и зациклиться
}
```
