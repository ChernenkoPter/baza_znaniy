Запрещает неявное преобразование типов через констурктор или оператор преобразования. Требует прямого вызова или использования **static_cast**
```
struct A { explicit A(int x) {} };
void f(A a) {}
f(10);        // ошибка: неявное преобразование запрещено
f(A(10));     // OK
f(static_cast<A>(10)); // OK
```
